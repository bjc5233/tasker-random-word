# tasker-random-word



## 说明
展示随机英语单词
* 安装tasker
* 导入tasker项目
* 配置根路径[word_root_path]变量为自己的路径，并在路径中放入[cet6.txt]和[md5.min.js]
* 配置按键音[word_sound_path]
* 配置样式风格[word_style]，可选有[透明trans\随壁纸color]
* 在桌面创建该任务的快捷方式
* 点击弹出随机一个英语单词；[→]为下一个随机英语单词；[x]为退出；[请输入单词]可手动指定查询单词
