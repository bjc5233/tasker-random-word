# tasker-random-word



## 说明
展示随机英语单词
* 安装tasker
* 导入tasker项目
* 变更[word_root_path]变量为自己的路径，并在路径中放入[cet6.txt]和[md5.min.js]
* 可配置界面风格[word_style]，可选值为[light, dark]
* 在桌面创建该任务的快捷方式
* 点击弹出随机一个英语单词；[→]为下一个随机英语单词；[↓]为退出；[请输入单词]可手动指定查询单词
